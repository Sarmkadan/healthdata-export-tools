# PerformanceUtility

`PerformanceUtility` is a static utility class designed to measure and report performance metrics for .NET applications. It provides methods to track execution time, memory usage, CPU utilization, and throughput calculations, enabling developers to profile and optimize code execution. The class is particularly useful for benchmarking operations, estimating progress, and diagnosing performance bottlenecks in data export or processing tasks.
